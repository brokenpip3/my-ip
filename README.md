# My-ip

Simple kubernetes deployment to simulate `ifconfig.co`,`ifconfig.me` or similar.

## Prerequisites

- K8s cluster
- Ingress controller(nginx or similar)

## Usage

`kubectl apply -f`

Change the ingress `host` fqdn according to your needs

`kubectl apply -f`
